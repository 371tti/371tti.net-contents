# [「「神と呼ばれ、魔王と呼ばれても」」](https://ncode.syosetu.com/n4845ec/)
圧倒的スケールと高解像度で描かれる複数主人公のファンタジー小説。　　
宗教、政治、戦争、社会、文化、経済、技術、魔法などの様々な要素が絡み合いながら物語が展開していく。　　
異なる時代を生きる主人公たちの人生が、歴史を通じて繋がっていくのと、主人公の人生に感動させられますよ？多分。　　

# [非科学的な犯罪事件を解決するために必要なものは何ですか？](非科学的な犯罪事件を解決するために必要なものは何ですか？)
異能力系ファンタジーで一番好きな小説。現代日本を舞台に、少女とおっさんが協力して、非科学的な犯罪事件を解決していく物語。  
テンポとキャラクターが良い。
