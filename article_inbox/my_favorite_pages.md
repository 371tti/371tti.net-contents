# 私の好きなWebページ集
私がよく見る、参考にしている、役に立った、面白いと思ったWebページのリストです。

## [私立PDD図書館](http://pddlib.v.wol.ne.jp/japanese/index.htm)
Public Domain Document(PDD)の資料を集めた図書館サイト。
百科辞書のデータベースを中心に、様々なジャンルの日本語の文書が収集されている。
よく利用させてもらってた

## [Rust プログラミング言語](https://doc.rust-jp.rs/book-ja/)
Rust言語の公式ドキュメントの日本語訳サイト。
Rustを学ぶ上で最も基本的なリソースであり、言語の基礎から応用まで幅広くカバーしている。
Rustを始めるならまずここから！

## [Rust 裏本(高度で危険な Rust Programming のための闇の技法)](https://doc.rust-jp.rs/rust-nomicon-ja/index.html)
この序論を読んだらもう読むしかありませんね★

> この本は、危険な Rust プログラムを正しく書くために理解しなくてはいけない、不愉快な詳細について詳しく見ていきます。このような性質上、この文書は今まで語られることのなかった恐怖を解き放ち、あなたの精神を何十億もの絶望のかけらに砕いてしまうかもしれません。  
> もし貴方が Rust とともに長く幸せな人生を歩みたいと望むなら、今すぐに背を向けて、この本を見てしまったことを忘れるのです。貴方には必要ないのですから。しかし、危険なコードを書く意思がもしも貴方にあるのなら、もしくはこの言語の最重要部にただ踏み込んでみたいのなら、この本は代えがたい情報をもたらすでしょう。  
> ...   
> はっきり言いますが、この本は詳細について深く説明します。例外の安全性、ポインタエイリアシング、メモリモデル、そして型理論についても少し。また、様々な種類の安全性や保証についてもたくさん説明します。  - [Rust 裏本](https://doc.rust-jp.rs/rust-nomicon-ja/ch01-00-introduction.html)

## [aquapal](https://aquapal.net/)
言わずもがなの有名サイト。
捨てアドや絵チャなどの秀逸なサービスを提供しているところ。
みんな大好きだよね！

## [とほほのWWW入門](https://www.tohoho-web.com/www.html)
とほほ師による様々な技術を解説したサイト。
非常に広く浅くまとまっていて、初心者から中級者まで参考になる内容が多い。

## [MDN Web Docs](https://developer.mozilla.org/en-US/)
Mozilla が提供するウェブ技術に関するドキュメントサイト。
HTML、CSS、JavaScript などのウェブ開発に関する情報が豊富に揃っている。
Web開発者にとっては欠かせないリソース。
